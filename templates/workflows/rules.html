About Workflow Automation
Triggers

  • Time-Based: After X days from an event

Actions

  • Send Email: Notify via email
  • Send Notification: Internal message
  • Create Task: Add follow-up task

  • Specific Job: Apply to one job only
  • Score Range: Min/max score requirements
