About Scheduled Tasks
Task Types
  • Job Closing Reminder: Notify when jobs are about to close
  • Interview Reminder: Send reminders before interviews
  • Application Follow-up: Remind to review applications
  • Document Reminder: Prompt applicants for missing documents
  • Report Generation: Generate periodic reports
  • Cleanup: Remove old data and logs
Schedule Types
  • Once: Run one time on a specific date
  • Daily: Run every day at specified time
  • Weekly: Run on specific day of week
  • Monthly: Run on specific day of month

Note: Tasks are processed by a background scheduler. Admins can manually trigger task processing from the Workflow Rules page.
