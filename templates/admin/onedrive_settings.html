Getting Started
Setup Instructions:
  1. Go to Azure Portal
  2. Navigate to "App registrations"
  3. Create a new application registration
  4. Copy the Tenant ID and Application ID
  5. Go to "Certificates & secrets"
  6. Create a new client secret
  7. Add API permissions:
    • Microsoft Graph → Files.ReadWrite.All
    • Microsoft Graph → Drive.ReadWrite.All
  8. Paste credentials above
Features:
  • Sync all documents to OneDrive
  • Sync per-applicant folders
  • Create shared links for documents
